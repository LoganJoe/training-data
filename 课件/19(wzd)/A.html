A. 大火车和小火车

统计
  • 描述
  • 提交
  • 自定义测试
  • 返回比赛

小w负责a国的火车系统调度。a国的火车系统由mm条铁轨构成，每条火车线路形如(a,b,c)(a,b,c)的形式，表示有两辆火车在cc时刻，分别从aa地和bb地出发前往对面。由于a国科技发达，我们可以认为火车不需要时间，在cc时刻出发就可以在cc时刻到达。

为了从xx到yy，小w可能会换乘一系列的铁路线路，小w为了从第一个铁路线路(a1,b1,c1)(a1,b1,c1)换乘第二个铁路线路(a2,b2,c2)(a2,b2,c2)，需要c1≤c2,b1=a2c1≤c2,b1=a2。

现在有qq个询问，每个询问问小w最早什么时间能从aa地到bb地。注意一条换乘线路的时间是线路上最后一段火车的到达时间。

输入格式

第一行三个整数n,m,qn,m,q。

接下来mm行，每行三个整数ai,bi,ciai,bi,ci。

最后qq行，每行表示一个询问ai,biai,bi, ai≠biai≠bi。

输出格式

qq行，每行一个整数表示答案。

如果不能到达输出-1.

样例一

input

3 3 3
1 3 1
2 3 2
1 2 1
1 2
1 3
2 3

output

1
1
1

样例二

input

3 3 3
1 3 5
2 3 2
1 2 7
1 2
1 3
2 3

output

7
5
2

数据范围

时间限制1s, 空间范围512MB

q≤105q≤105
